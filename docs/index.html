Abyss is a desktop Minecraft launcher with a Modrinth browser built in. Launch any version and loader, install mods and whole modpacks in a click, and keep your worlds in tidy, separate instances.

Launch any version

Every Minecraft release and snapshot, with Fabric, Quilt, NeoForge and Forge loaders.

⌕

Modrinth, built in

Search the full Modrinth catalogue and drop mods straight into an instance in one click.

▣

One-click modpacks

Install a whole .mrpack and Abyss builds a matching instance with every mod and config.

◈

Tidy instances

Separate versions, mods and saves per instance, each with its own memory allocation.

⬡

Real Microsoft login

Sign in once with your Microsoft account; the session refreshes itself after that.

◎

Friends

Keep a friends list of real Minecraft accounts, checked against Mojang with their skins.

Get Abyss

Grab the latest Windows build — an installer or a portable zip, whichever you prefer.

▼ Download for Windows Portable .zip

The build isn't code-signed yet, so Windows SmartScreen may warn you — click “More info” then “Run anyway”. You'll need Java 17+ and a Microsoft account that owns Minecraft: Java Edition to actually launch the game.
